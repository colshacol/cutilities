cutilities

Colton's Utility Classes

My name is Colton. I am a novice web developer from central Texas. I created cutilities as a learning project that I never thought anyone in their right mind would ever use seriously. Nonetheless; I am proud of my cutilities.

Color can be added to your element by giving it a class like this: class="colr000000" Use any hexidecimal value you want and cutilities will automatically apply color: #HEXCODE to your element.

Margins and paddings are easily defined using custom variations of mr25 mt25 mlau mrau, or the padding equivalent pt55pr90pb10plau. It doesn't matter what order you use the classes in, nor if they are sitting right next to each other. As long as you have the class pt65 in your element's class, you'll have padding-top: 65px applied.
